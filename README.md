# Simple Basic Web


各種技術を学習するためのシンプルな基本Webアプリのコードになります。


### このコードで最低限身につけるべきスキル・知識

- HTML/CSS/JavaScriptの役割を説明できるようになる。
  - HTMLとは何ですか？
  - CSSとは何ですか？
  - JavaScriptとは何ですか？
- 正規表現とはなんですか？
  - 各正規表現の意味を説明してください。
  - できればEmailの正規表現をもっと高精度に変更して下さい。

- 備考
　- できれば、Simple Basic Webくらいは空で実装できるようにしておく。

### 技術学習例

- フォームが実際に動作するようにAPIを実装する。
- React/Next.jsを利用して実装してみる。
- ReactNative（expo）を利用してアプリ化してみる。
- などなど。
