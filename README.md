# Simple Basic Web


各種技術を学習するためのシンプルな基本Webアプリのコードになります。
実際に動くものは[こちら](https://eizaburo.github.io/simple-basic-web/index.html)で確認できます。

> Webの理解はもちろんですが、最終的にはシステム開発全般の学習の利用を想定した仕様になっています。

### コンテンツの利用方法

静的/クライアントのみで動作するWebアプリなので、cloneして、index.htmlを開いてください。
Webアプリの動作確認自体は、[Gighub Pages上に展開してあるもの](https://eizaburo.github.io/simple-basic-web/index.html)で確認できます。

### コンテンツの活用方法

まずは **「空」で、記述できるようになるまで記述してみてください**。
その上で、以下の質問や課題にスムーズに対応できるようになって下さい。

> 今は無理でも、そうなるためのコンテンツを随時提供予定です。


#### 基礎スキル・知識の確認

以下の質問に正確に答えて下さい。

- HTMLとは何ですか？
- CSSとは何ですか？
- MediaQueryとは何ですか？
- JavaScriptとは何ですか？
- 正規表現とは何ですか？

#### 応用スキル・知識の確認

以下の対応をしてみて下さい。

##### 技術系

- 実際に問合せを受けつけるAPIを実装して下さい。
  - 当社では、API実装にnode.jsを、DBにはPostgreSQLを利用することが多いです。
- 同じ仕様のWebアプリをReactで実装してください。
- 同じ仕様のWebアプリをNext.jsで実装してください。
- WebアプリおよびAPIの妥当なデプロイ先を選定し、その理由を資料化し説明して下さい。

##### PM系

- Webアプリの仕様書（要件定義書・設計書）を作成してください。
- 本Webアプリ開発の見積を作成し、その内容の妥当性を説明して下さい。

#### トレンド・AI系

- 同じ仕様のWebアプリをAIで生成してみて下さい。
- 同じ仕様のアプリになっているか、検証してエビデンスを提示して下さい。

### 備考

非常に簡素なサンプルWebアプリですが、上記に対応するのは以外と難しいものです。
